Terms and Conditions

This application is a clinical decision support prototype intended for evaluation and demonstration purposes only.

The application accesses patient data from the Epic electronic health record via SMART on FHIR to provide non-diagnostic decision support related to pulmonary embolism (PE) rule-out.

  • This application does not diagnose disease.
  • This application does not place, modify, or cancel orders.
  • This application does not replace clinician judgment.

Patient data accessed by this application is used only for real-time decision support and is not stored outside the health system environment except as required for application functionality.

Use of this application is subject to institutional approval and applicable privacy and security policies.
